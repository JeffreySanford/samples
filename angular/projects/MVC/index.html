AngularJS Architecture Patterns


AngularJS relies on two key architecture patterns:

  • Model-View_controller (MVC)
  • Model-View-ViewModel($scope) (MVVM)

The combination for these two patterns could be refered to as MV*. Pattern provide prescritive guidance to developers that can be used to build organizaed applications.
